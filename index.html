Kết nối GitHub

Chỉ cần điền một lần trên máy bạn. Token lưu trong trình duyệt của bạn, không bị đăng lên web.

Lấy token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Chọn đúng repo chứa web, phần Repository permissions → Contents đặt Read and write. Dán token vào ô bên dưới. Chỉ thực hiện trên máy tính cá nhân — không đăng nhập/dán token trên máy công cộng.
